# 网站制作文案脚本：《听读小站》

## 1. 网站定位

"听读小站"是一个为高中生精选的英语阅读与听力资源导航网站。目标是通过优质、趣味性的内容，帮助学生提升词汇量、阅读和听力水平。

## 2. 网站整体风格

- **主色调**：蓝白简洁风
- **功能按钮（右上角）**：
  - Light/Dark 模式切换（白底 ↔ 黑底）
  - 中英语言切换（所有导航/介绍文本可切换，英文资源名称保持不变）
- **卡片风格**：正方形卡片，封面图 + 简短介绍，点击进入详情页（详情页有更详细介绍 + 外部链接）。

## 3. 网站结构

### 首页（Home）

- **顶部简短介绍**：
  "欢迎来到听读小站！这是一个帮助高中生提升英语阅读与听力的精选资源平台。你可以通过测试了解自己的水平，再找到适合的资源。"

- **核心按钮/链接**：
  - 词汇量测试：https://my.vocabularysize.com/
  - 阅读/听力水平测试：https://www.efset.org/
  - 跳过测试，直接浏览资源（跳转到资源页面）

- **底部**：链接到"关于本站"（作者学习经历 & 参考文献）。

### 资源页面（Resources）

- **筛选功能**：
  - 按类型：阅读 / 听力
  - 按难度：Elementary / Lower-Intermediate / Intermediate / Upper-Intermediate / Advanced
  - 搜索框：支持关键字搜索资源

- **展示方式**：
  - 卡片包含：封面图、资源名称、标签（tags）、简短介绍
  - 点击卡片进入详情页：更详细介绍 + 外部链接

## 5. 技术功能需求

- Light/Dark 模式切换（右上角按钮）
- 中英语言切换（右上角按钮，除英文资源名称外）
- 资源筛选系统（前端筛选，无需后台数据库）
- 搜索框（关键字匹配资源标题/标签/介绍）
- 静态网站（可通过更新数据文件新增资源，初期不需要动态后台）
